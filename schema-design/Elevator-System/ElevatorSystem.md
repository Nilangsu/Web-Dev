# Smart Elevator Control

This database is designed for a Smart Elevator Control System. It supports:

- Buildings which contain buildng details
- Elevators to contain various elevator data from different elevators in the building
- Elevator stuffs which contain data of stuffs who look after the elevator system
- Floors data and access from which elevators can know if a floor is accessible.
- Elevator shafts to put min and max limit of an elevator
- Ride assignments, floor requests to assign elevators based on request to a person when they request
- Ride logs to contain data when a requestion is made, when it's handled and which floor the person wants to stop in
- Floor stops to contain multiple stops data as an elevator might stop more than once while a request is on going
- Elevator status tracking to track various status of the elevator in normal conditions as well as it's scheduled time of running and while maintenance
- Maintenance tracking to keep records of maintenance as well as data of stuffs who is doing the maintenance
- Alerts system in case of emergency and to inform stuffs
